# Acuvera - AI-Powered Medical Billing Intelligence

A world-class AI SaaS website built with Next.js, Three.js, and Framer Motion. Features premium 3D animations, interactive product demos, and enterprise-grade design.

![Tailwind](https://img.shields.io/badge/Tailwind-3.4-38B2AC?style=flat-square&logo=tailwind-css)

## Features

- **3D Animations** - Floating particles and interactive document transforms using Three.js
- **Smooth Animations** - Physics-based motion with Framer Motion
- **Interactive Demo** - Live product simulation showing the 3-stage AI pipeline
- **Premium Design** - Enterprise-grade UI inspired by Stripe, OpenAI, and Anthropic
- **Responsive** - Fully responsive across all devices
- **Performance** - GPU-accelerated animations, optimized bundle size

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS + CSS Variables
- **Animations:** Framer Motion + GSAP
- **3D:** React Three Fiber + Three.js
- **Icons:** Lucide React
- **Build:** Static export optimized for deployment

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Deployment

The site is configured for static export. The build output is in the `dist` directory.

## AI Pipeline

The website showcases Acuvera's 3-stage AI pipeline:

1. **Claude 3.5 Sonnet** - Document extraction and understanding
2. **BioBERT + PyCTAKES** - Medical code validation
3. **MedGemma** - Clinical reasoning via Vertex AI
